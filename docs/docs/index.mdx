# デジタルサイネージ ユーザーガイド

このドキュメントは、Raspberry Pi 向けデジタルサイネージシステムの操作方法をまとめたユーザーガイドです。ダッシュボードの基本操作からコンテンツ管理、サイネージ表示の設定まで、各機能の使い方を説明します。

## ガイド一覧

### [ログインと認証](/login)

ダッシュボードへのログイン・ログアウト手順、初回パスワード設定、パスワードを忘れた場合の対処方法を説明します。

### [コンテンツアップロード](/upload)

画像（PNG/JPEG）や動画（MP4/MOV/WMV）をサイネージに表示するためのアップロード手順を説明します。

### [コンテンツ管理](/manage-contents)

アップロード済みコンテンツの並び替え、表示/非表示の切り替え、表示時間の変更、削除などの管理操作を説明します。

### [表示画面調整](/view-position)

Raspberry Pi に接続されたディスプレイ上でのコンテンツ表示位置やサイズの調整方法を説明します。

### [エリア管理（管理者向け）](/area-management)

管理者向けのエリア追加・編集・削除の操作方法を説明します。

### [アカウント一覧管理（管理者向け）](/user-account-management)

管理者向けのユーザーアカウント作成・編集・削除、権限設定の操作方法を説明します。

### [アカウント詳細管理](/account-setting)

自分のアカウント情報の確認、ユーザー名やパスワードの変更方法を説明します。

### [サイネージ表示画面](/signage-display)

Raspberry Pi 上でサイネージ表示画面を起動し、コンテンツをスライドショー表示する方法を説明します。

### [ダッシュボード共通操作](/common-operations)

サイドバーメニューの操作、エリア選択、管理者と一般利用者のメニュー表示の違いなど、共通的な操作方法を説明します。
