# サイネージ表示画面

Raspberry Pi 上でサイネージ表示画面を起動し、コンテンツを表示する方法を説明します。サイネージ表示画面では、登録済みのコンテンツがスライドショー形式で自動的に切り替わりながら表示されます。

## サイネージ表示画面へのアクセス

サイネージ表示画面は、ブラウザで以下の URL にアクセスして表示します。

```
http://<サーバーのアドレス>/?areaId=<エリアID>
```

- `<サーバーのアドレス>` — サイネージシステムが稼働しているサーバーのホスト名またはIPアドレスとポート番号（例: `localhost:3000`）
- `<エリアID>` — 表示したいエリアのIDを指定する

**アクセス手順:**

1. Raspberry Pi のブラウザを開く
2. アドレスバーに `http://<サーバーのアドレス>/?areaId=<エリアID>` を入力する
3. Enter キーを押すと、指定したエリアのサイネージ表示画面が表示される

![サイネージ表示画面](./images/signage-display-screen.png)

:::tip
エリアIDは、ダッシュボードのエリア管理画面で確認できます。`areaId` を省略した場合はデフォルトのエリアが表示されます。
:::

## スライドショーによる自動切り替え表示

サイネージ表示画面にアクセスすると、登録済みのコンテンツがスライドショー形式で自動的に切り替わります。

- コンテンツは「コンテンツ管理」画面で設定した並び順に従って表示される
- 各コンテンツは設定された表示時間が経過すると、自動的に次のコンテンツに切り替わる
- すべてのコンテンツの表示が完了すると、ページが自動的にリロードされ、最初のコンテンツから再び表示が開始される
- 表示中のコンテンツは画面全体に黒い背景で表示され、コンテンツ間の切り替えはフェード効果で行われる

## 画像コンテンツと動画コンテンツの混在表示

サイネージ表示画面では、画像コンテンツと動画コンテンツを混在させて表示できます。

### 画像コンテンツの表示

- 画像（PNG/JPEG）は「コンテンツ管理」画面で設定した表示時間（秒）の間、画面に表示される
- 表示時間が経過すると、自動的に次のコンテンツに切り替わる

### 動画コンテンツの表示

- 動画（MP4/MOV/WMV）は自動的に再生が開始される
- 動画は音声なし（ミュート）で再生される
- 動画の表示時間は「コンテンツ管理」画面で設定された時間に基づいて切り替わる

### 表示順序の例

「コンテンツ管理」画面で以下の順序に設定した場合:

1. 画像A（表示時間: 5秒）
2. 動画B（表示時間: 動画の長さに基づく）
3. 画像C（表示時間: 10秒）

サイネージ表示画面では、画像A → 動画B → 画像C の順に自動で切り替わり、画像C の表示後にページがリロードされて画像A から再び表示されます。

## 表示画面調整の反映

「表示画面調整」画面で設定した画像のサイズ・位置は、サイネージ表示画面に自動的に反映されます。

反映される設定項目:

- **サイネージ画像（高さ）** — コンテンツの表示高さ
- **サイネージ画像（幅）** — コンテンツの表示幅
- **サイネージ上部余白** — 画面上端からコンテンツまでの余白
- **サイネージ左部余白** — 画面左端からコンテンツまでの余白

表示画面調整が未設定の場合は、ブラウザのウィンドウサイズに合わせて全画面表示されます。

:::note
表示画面調整の設定を変更した場合、サイネージ表示画面をリロードすると新しい設定が反映されます。
:::
